# Design System

## 1. Typography
* **Primary Font:** Inter, Outfit
* **Headings:** Outfit (Bold, tracking tight)
* **Body Text:** Inter (Regular/Medium)

## 2. Color Palette
* **Primary (Brand):** Indigo-600 (`#4f46e5`) / Violet-600 (`#7c3aed`)
* **Secondary:** Cyan-500 (`#06b6d4`)
* **Dark Mode Background:** Slate-950 (`#020617`)
* **Surface Background:** Slate-900 (`#0f172a`) / Slate-800 (`#1e293b`)
* **Text Primary:** Slate-50 (`#f8fafc`)
* **Text Secondary:** Slate-400 (`#94a3b8`)

## 3. UI Patterns
* **Glassmorphism:** `backdrop-filter: blur(12px) saturate(180%); background-color: rgba(15, 23, 42, 0.6); border: 1px solid rgba(255, 255, 255, 0.1);`
* **Micro-animations:** Smooth ease-in-out transitions on buttons, inputs, and interactive cards (e.g., scale on hover, soft glow effects).
