# N.O.V.A Platform Backend

FastAPI asynchronous backend for the N.O.V.A learning platform.

## Features

- **Asynchronous Architecture**: Powered by FastAPI, Uvicorn, and async SQLAlchemy.
- **Database Migrations**: Managed via Alembic.
- **Containerized**: Integrated with Docker and Docker Compose.
- **Repository Pattern**: Structured data access layer.

## Setup and Running

To run locally using Docker Compose, execute the following command in the project root directory:

```bash
docker compose up --build
```
